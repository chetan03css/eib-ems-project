Welcome To Employee Info Book

Employee Info Book is a web application, which enables you to lookup the employees in the organisation.You can add,update,delete and find the employees details and useful reports at once. This web app is Simulation of a typical Employee Information System !

Services

  • View Employee Data
  • Add a new employee !
  • Update any employee's details !
  • Delete/Remove an employee record !
  • Get Useful Reports !

About

Employee Information Systems help us manage employee details in an organisation. When any employee joins the organisation, his/her data, personal and job related, are stored in a database. This application runs on the web server which can be accesed by the users connected to the internet. Users can find various services of this app, at "Tasks" menu.

This app is a simulator of employee information system. We have named it "Employee Info Book", where in you can manage the employees. The design of the this site is shown as a notebook, which symbolizes our info book. This application is built using "HTML","CSS","JAVA SCRIPT","JSP","JSTL" technologies.
